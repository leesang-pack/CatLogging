## Quick start
1. start
```
    java -jar catlogging.war
```

2. Option Start
```
    java -Dcatlogging.validationPath=/ -Dcatlogging.home=/home/catlogging -Dlog4j.configuration=log4j-dev.properties -jar catlogging.war
```
